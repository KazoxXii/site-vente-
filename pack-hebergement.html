C'est quoi un nom de domaine ?

C'est l'adresse de votre site web. Comme une adresse postale, mais sur internet. Par exemple : pikbowl.com ou leqgb.fr.

Sans nom de domaine, les clients ne peuvent pas vous trouver. C'est la première chose qu'il faut acheter pour avoir un site web.

✓ .fr — pour une entreprise en France
✓ .com — international, le plus connu
✓ .net — alternative si .com est pris
✓ .restaurant — spécifique à votre métier
🏠

C'est quoi l'hébergement ?

C'est l'endroit où votre site est stocké. Un serveur (un ordinateur puissant) tourne 24h/24 pour afficher votre site à chaque visiteur.

Sans hébergement, votre site n'existe nulle part. C'est comme avoir une boutique sans local — le nom de domaine c'est l'enseigne, l'hébergement c'est le local.

1
Le nom de domaine → c'est l'adresse (pikbowl.fr)
2
L'hébergement → c'est le local (le serveur qui stocke les fichiers)
3
Le site web → c'est la boutique (les pages que les clients voient)
📧
