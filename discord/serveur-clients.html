🚀 Comment créer le serveur

Étape 1 — Créer le serveur

Ouvrir Discord → Cliquer "+" → "Créer un serveur" → Nommer : MALTY Clients

Étape 2 — Créer les rôles

Paramètres du serveur → Rôles → Créer :

👑 Admin → Couleur rouge → Permissions : Tout 👤 Client → Couleur bleue → Permissions : Voir/écrire dans son dossier 📦 Archivé → Couleur grise → Permissions : Lecture seule

Étape 3 — Créer les catégories et channels

Suivre la structure ci-dessus. Pour chaque nouveau client :

1. Créer un channel dans "📄 DOSSIERS CLIENTS" → nom : client-nom-site 2. Assigner le rôle "👤 Client" au membre 3. Limiter la visibilité du channel à ce rôle uniquement

Étape 4 — Inviter les clients

Paramètres du serveur → Inviter → Copier le lien → L'envoyer au client par email

Le client rejoint, voit uniquement son dossier et les channels généraux.

💡 Bonnes pratiques

Pour MALTY

• Un channel par client, jamais de mélange
• Les identifiants vont dans #identifiants (privé admin)
• Les sites terminés vont dans #terminés puis #archivés
• Nettoyer les anciens fichiers régulièrement

Pour les clients

• Déposer tous les fichiers dans votre channel dédié
• Nommer les fichiers clairement : logo-v2.png, textes-accueil.docx
• Poser les questions dans votre channel
• Ne pas partager le lien d'invitation
