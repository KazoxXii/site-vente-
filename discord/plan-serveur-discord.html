📋 Étapes de création

1

Ouvre Discord

Clique sur "+" à gauche puis "Créer un serveur"

2

Choisis le template

"Communauté" ou "Créer le mien"

3

Nom du serveur

MALTY — Création de Sites Web

4

Icône du serveur

Upload le fichier logo.svg ou logo-social.svg

5

Crée les channels et rôles

Suis le plan ci-dessous

🏷️ Rôles à créer

Paramètres du serveur → Rôles → Créer un rôle

👑 Fondateur ⚡ Admin 💎 VIP 🎨 Client 👤 Membre

Permissions par rôle

👑 Fondateur : Toutes les permissions
⚡ Admin : Gérer les messages, kick, ban
💎 VIP : Accès channels exclusifs, vocal
🎨 Client : Accès channel projet, support
👤 Membre : Lecture seule + écriture général

⚙️ Configuration importante

Paramètres du serveur

→ Verification Level : Moyen
→ Notification par défaut : Nothing
→ Canal d'annonce : #annonces
→ Ajouter une bannière : Image MALTY
→ Lien d'invitation : Génère un lien permanent

Auto-rôles (optionnel)

Utilise un bot comme MEE6 ou Carl-bot pour attribuer automatiquement le rôle "👤 Membre" aux nouveaux arrivants.

Bots recommandés

→ MEE6 : Auto-rôle, modération, levels
→ Carl-bot : Réactions, logs, automod
→ Ticket Tool : Système de tickets support
→ Statbot : Statistiques du serveur

📝 Règlement à copier-coller

📋 **RÈGLEMENT DU SERVEUR MALTY** 1️⃣ **Respect mutuel** — Pas d'insultes, de harcèlement ou de discrimination. 2️⃣ **Pas de spam** — Un message par message, pas de flooding. 3️⃣ **Pas de pub non autorisée** — Demande avant de poster un lien. 4️⃣ **Utilise les channels** — Chaque topic a son channel. 5️⃣ **Pas de contenu NSFW** — Zéro tolérance. 6️⃣ **Respecte le staff** — Les décisions sont finales. 7️⃣ **Amuse-toi bien** — On est là pour apprendre et échanger ! 🎯 ⚠️ Non-respect = Avertissement → Mute → Kick → Ban
